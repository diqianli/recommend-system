PinSage - Graph Convolutional Neural Networks

图卷积推荐网络 | Pinterest | KDD 2018

图神经网络 随机游走 大规模图 视觉推荐

算法概述

PinSage是Pinterest在2018年KDD上提出的图神经网络推荐模型，是首个应用于十亿级规模的图神经网络推荐系统。它通过随机游走采样和图卷积，为Pinterest的30亿Pin学习高质量的嵌入表示。

核心贡献：将图神经网络扩展到工业级规模(30亿节点、180亿边)，同时保持高效的训练和推理。

核心技术

1. 随机游走采样

从目标节点出发随机游走，统计访问频次选择Top-K重要邻居

2. 图卷积

聚合邻居特征，通过多层卷积学习节点表示

3. 生产者-消费者模式

GPU计算嵌入，CPU采样邻居，并行高效

4. MapReduce推理

分布式计算所有节点嵌入

随机游走采样

采样流程
目标Pin
→ Board → Pin → Board ×12
→ Board → Pin → Board ×8
→ Board → Pin ×5
Top-K邻居: Pin1 Pin2 Pin3 ...
优势：不需要对全图做卷积，只采样重要邻居，计算量从O(全图)降到O(K)

图卷积操作

PinSage 图卷积公式

步骤1: 聚合邻居
nu = Σv∈N(u) hv / |N(u)|

步骤2: 结合自身
hunew = ReLU(W · [hu || nu])

步骤3: 归一化
hunorm = hunew / ||hunew||

优缺点分析

优点

  • 首个十亿级GNN推荐系统
  • 随机游走采样高效
  • 图结构信息充分利用
  • 多模态特征融合
  • 工业验证效果显著

局限性

  • 随机游走有随机性
  • 需要大量工程优化
  • 训练资源消耗大
  • 推理需要全图计算
