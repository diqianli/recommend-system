DIN - Deep Interest Network

深度兴趣网络 | 阿里巴巴 | KDD 2018

排序算法 注意力机制 用户兴趣建模

算法概述

DIN是阿里巴巴在2018年KDD上提出的深度学习推荐模型，核心创新是使用注意力机制来捕捉用户历史行为中与当前候选物品相关的兴趣表达。

核心思想：用户的兴趣是多样化的，对于不同的候选物品，用户历史行为中的相关部分应该被赋予不同的权重。

局部激活单元

设计注意力机制，根据候选物品自适应激活用户历史行为中的相关部分

变长兴趣表达

用户兴趣向量长度随历史行为变化，而非固定长度

Dice激活函数

提出Data Adaptive Activation Function，改进PReLU

小批量正则化

提出Mini-batch Aware Regularization，解决稀疏特征过拟合

核心公式

注意力权重计算

αi = f(Embcandidate, Embi)

其中 f 是一个前馈神经网络，输出归一化后的注意力权重

用户兴趣向量

Vu = Σ αi · Embi

加权求和得到自适应的用户兴趣表示

与基线模型对比

传统 Embedding+MLP

✗ 固定长度用户向量
✗ 历史行为求和/平均
✗ 无法区分兴趣相关性

DIN 深度兴趣网络

✓ 自适应兴趣表达
✓ 注意力加权聚合
✓ 对候选物品敏感

优缺点分析

优点

  • 自适应捕捉用户兴趣
  • 注意力权重可解释性强
  • 工业验证效果显著
  • 提出的技术具有通用性

局限性

  • 未考虑用户兴趣的时序演化
  • 历史行为之间相互独立
  • 长序列计算开销大
