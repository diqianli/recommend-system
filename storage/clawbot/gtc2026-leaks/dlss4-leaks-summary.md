### 4.2 帧生成技术原理

```
┌────────────────────────────────────────────────────────────────────┐
│                    帧生成技术原理                                    │

│                                                                     │

│                                                                     │
│  DLSS 4.5 多帧生成:                                                 │

│  (AI在每个真实帧后生成多个中间帧)                                    │
│                                                                     │
└────────────────────────────────────────────────────────────────────┘
```

### 4.3 延迟优化

DLSS 4.5 引入了动态延迟优化技术：

- **NVIDIA Reflex 集成**: 减少系统延迟
- **动态帧生成**: 根据场景复杂度调整生成帧数
- **预测算法**: 提前计算下一帧可能的内容
