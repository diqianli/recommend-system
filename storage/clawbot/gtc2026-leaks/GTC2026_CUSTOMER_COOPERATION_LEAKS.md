## 1. OpenAI-NVIDIA 战略合作

### 1.1 10GW AI数据中心合作 (2025年9月宣布)

| 项目 | 详情 |
|------|------|
| **合作规模** | 至少 10 吉瓦 (10 GW) NVIDIA 系统 |
| **GPU数量** | 数百万颗 GPU |
| **NVIDIA投资** | 高达 1000 亿美元 (原计划) |
| **首批部署** | 2026年下半年 - Vera Rubin 平台 |
| **合作性质** | 战略合作伙伴, 首选计算和网络供应商 |

### 1.2 最新投资动态 (2026年2-3月)

| 日期 | 事件 | 金额 |
|------|------|------|
| 2026年2月 | 原定1000亿美元计划被取消 | - |

| 2026年3月4日 | 黄仁勋评论: "可能是最后一次投资" | - |

### 1.3 合作要点

- OpenAI 将 NVIDIA 作为其 AI 工厂增长计划的**首选战略计算和网络合作伙伴**
- 双方将**共同优化路线图** - OpenAI的模型/基础设施软件与NVIDIA的硬件/软件
- 合作补充了与 Microsoft、Oracle、SoftBank 和 Stargate 合作伙伴的现有合作
- OpenAI 目前拥有超过 **7亿周活跃用户**

## 2. Microsoft-NVIDIA 深度合作

### 2.1 Azure Rubin 部署准备

| 项目 | 详情 |
|------|------|
| **部署平台** | NVIDIA Vera Rubin NVL72 |
| **部署地点** | Fairwater 数据中心 (威斯康星州、亚特兰大) |
| **性能规格** | 每机架 3.6 EF NVFP4 (较GB200提升5倍) |
| **部署时间** | 2026年下半年 |

### 2.2 Azure 基础设施准备

Microsoft Azure 已为 Rubin 平台做好了以下准备:

- **NVLink 第六代**: 支持 ~260 TB/s 规模扩展带宽
- **ConnectX-9 网络**: 1,600 Gb/s 超快网络
- **HBM4/HBM4e 内存**: 升级冷却和功率设计
- **SOCAMM2 内存扩展**: 新架构已集成验证
- **多芯片封装**: 供应链和机械设计已预调

### 2.3 历史部署里程碑

| 代际 | 部署内容 |
|------|----------|
| Ampere/Hopper | 全球最大商业 InfiniBand 部署 |
| GB200 NVL72 | 首个也是最大规模实现 |
| GB300 NVL72 | 首个也是最大规模实现 |

### 2.4 设计差异化优势

- **Pod 交换架构**: GPU服务器托盘可快速更换
- **冷却抽象层**: 支持Rubin多芯片高带宽组件
- **新一代功率设计**: 液冷循环、CDU扩展、高安培母线
- **AI超级工厂模块化**: 区域级超级计算机而非单一巨型站点

### 2.5 数据来源

- [Microsoft Azure 官方博客](https://azure.microsoft.com/en-us/blog/microsofts-strategic-ai-datacenter-planning-enables-seamless-large-scale-nvidia-rubin-deployments/)
