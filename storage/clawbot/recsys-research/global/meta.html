News Feed | 广告推荐 | Reels | DLRM

系统概览

Meta(原Facebook)是全球最大的社交媒体公司，其推荐系统支撑着Facebook News Feed、Instagram Feed、Reels、广告系统等产品。Meta在深度学习推荐系统领域有开创性贡献，DLRM模型已成为行业标准架构。

核心算法

DLRM (Deep Learning Recommendation Model)

DLRM架构

Meta提出的经典推荐模型架构：

┌─────────────────────────────────────────┐
│  稀疏特征 (类别特征)                     │
│  → Embedding Table (数十亿参数)          │
│  → Pooling                               │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│  特征交叉 (Feature Interaction)         │
│  - 点积交叉                               │
│  - 所有Embedding两两交互                 │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│  稠密特征 + 交叉特征                      │
│  → MLP (多层感知机)                      │
│  → 输出概率                              │
└─────────────────────────────────────────┘

来源: DLRM: An Industry-Class Deep Learning Recommendation Model, arXiv 2019

多任务学习

多目标推荐

Meta推荐系统同时优化多个目标：

  • 点击率 (CTR)
  • 互动率 (点赞、评论、分享)
  • 停留时长
  • 转化率 (广告)
