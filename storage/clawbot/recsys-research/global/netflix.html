系统架构

用户打开Netflix
    ↓
┌────────────────────────────────────────┐
│  上下文获取                             │
│  - 用户观看历史                         │
│  - 时间/设备                            │
│  - 当前会话行为                         │
└────────────────────────────────────────┘
    ↓
┌────────────────────────────────────────┐
│  召回层                                 │
│  - 协同过滤                             │
│  - 内容相似度                           │
│  - 热门/新品                            │
└────────────────────────────────────────┘
    ↓
┌────────────────────────────────────────┐
│  排序层                                 │
│  - 深度学习模型                         │
│  - 个性化预测                           │
└────────────────────────────────────────┘
    ↓
┌────────────────────────────────────────┐
│  页面生成                               │
│  - 行生成 (Trending, Because You...)   │
│  - 封面选择                             │
│  - 排序优化                             │
└────────────────────────────────────────┘

架构特点

  • 深度个性化: 每个用户看到的首页都不同
  • A/B测试文化: 所有变更都经过严格测试
  • 封面个性化: 同一内容不同封面
  • 批量+实时: 离线计算+实时响应
